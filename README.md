# AI Learning Platform

這是一個互動式的AI學習平台，提供個性化的學習體驗和即時反饋。

## 功能特點

- 個人化學習評估
- AI學習助手
- 課程內容管理
- 學習進度追蹤
- 即時學習分析

## 技術架構

- 前端：HTML5, CSS3, JavaScript
- AI服務：自然語言處理, 學習模式分析
- 資料分析：學習行為追蹤, 進度分析

## 本地開發

1. 克隆倉庫
```bash
git clone https://github.com/glen200392/ai-learning-platform.git
```

2. 打開項目
```bash
cd ai-learning-platform
```

3. 在瀏覽器中打開
- 直接打開 index.html
- 或使用本地服務器運行
